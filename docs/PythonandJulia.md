# Python and Julia

PyNOF is a version of DoNOF written in Python. You can install it with
```{margin}
![PyNOF](./PyNOF.png)
```
~~~
pip install pynof
~~~

Similarly, DoNOF.jl is a version of DoNOF written in the Julia language. You can install DoNOF.jl form repl with
```{margin}
![DoNOF.jl](./DoNOFjl.png)
```
~~~
add DoNOF
~~~
